Python AST Viewer

Interactive AST (syntax tree) viewer for Python code.
Enter your code — get the structure visualization.
The app runs entirely in your browser.
Requires downloading approximately 70–100 MB during startup (Pyodide and dependencies).
A stable internet connection is recommended. Your code is not sent to any server.
Features:
  • Parsing and visualization of AST for any Python code
  • Highlighting of AST source fragments
  • Works fully locally and anonymously
